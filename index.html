📱 如何在手机上练习？

  3. 手机浏览器打开 cmd 里显示的地址，如 http://192.168.x.x:8080

  5. 已部署 GitHub Pages 时：直接打开 https://用户名.github.io/仓库名/
  6. iPhone：分享 →「添加到主屏幕」；安卓：菜单 →「添加到主屏幕」
